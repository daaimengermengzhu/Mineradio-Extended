# 问题与风险

## 风险记录

- 酷狗播放地址接口可能因会员、付费、版权、地区或官方客户端限制不返回 URL。
- `.kugou-cookie` 是本地私有状态，不能提交到 Git。
- 酷狗接口字段可能随上游变化，需要保持映射逻辑保守。
- 当前只接入酷狗搜索和播放地址，酷狗歌单、歌词、收藏、红心同步还未接入。

## 失败尝试

- 首次真实调用 `/api/kugou/search` 返回 500，原因是本地 cookie 中的中文昵称被直接放进 HTTP `Cookie` 请求头；已改为发酷狗 API 时只带必要的 ASCII auth/device 字段。
