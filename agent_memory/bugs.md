## 风险记录

- 酷狗播放地址接口可能因为会员、付费、版权、地区或官方客户端限制不返回 URL。
- `.kugou-cookie` 是本地私有状态，不能提交到 Git。
- 酷狗接口字段可能随上游变化，需要保持映射逻辑保守。
- 综合搜索里如果用户点击的是 `NE` 或 `QQ` 标签结果，酷狗登录不会影响那条歌曲的权限；应优先点击 `KG` 标签结果或切换到 `KG` 搜索模式验证。
