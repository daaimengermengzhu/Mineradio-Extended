# 问题与风险

## 已知问题

- 酷狗概念版 VIP 状态目前仍主要来自本地会话解析，尚未接入可信的会员详情接口。
- 酷狗新建酷狗歌单还未接入。

## 风险记录

- 酷狗播放地址接口可能因为会员、付费、版权、地区或官方客户端限制不返回 URL。
- `.kugou-cookie` 是本地私有状态，不能提交到 Git。
- 酷狗接口字段可能随上游变化，需要保持映射逻辑保守。
- 综合搜索里如果用户点击的是 `NE` 或 `QQ` 标签结果，酷狗登录不会影响那条歌曲的权限；应优先点击 `KG` 标签结果或切换到 `KG` 搜索模式验证。
- 酷狗收藏到歌单属于真实账号写入操作，本轮只做了非写入接口验证；需要用户在 App 里点一次收藏确认真实写入效果。
- 酷狗红心属于真实账号写入操作，本轮只做了状态读取和非写入接口验证；需要用户在 App 里点一次 KG 心形按钮确认真实新增效果。
- 酷狗歌词旧版 `https://lyrics.kugou.com/search` 当前可返回 `id/accesskey`，参考项目中的 `/v1/search` 当前环境返回空内容，因此保留 `/v1/search` 作为备用而不是主路径。

## 失败尝试

- 首次真实调用 `/api/kugou/search` 返回 500，原因是本地 cookie 中的中文昵称被直接放进 HTTP `Cookie` 请求头；已改为发酷狗 API 时只携带必要的 ASCII auth/device 字段。
- Playwright 默认 Chromium 缺失，改用本机已安装 Chrome 完成浏览器验证。
