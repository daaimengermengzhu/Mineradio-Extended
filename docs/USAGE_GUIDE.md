# Mineradio 扩展版使用教程

这份教程面向普通用户，默认你只想下载安装包并开始听歌，不需要会 Git 或命令行。

## 1. 下载和安装

1. 打开最新版下载页：[Mineradio-Extended Releases](https://github.com/daaimengermengzhu/Mineradio-Extended/releases/latest)。
2. 在 `Assets` 区域下载 `Mineradio-1.1.1-Setup.exe`。
3. 双击安装包，按提示安装。
4. 安装完成后，从桌面快捷方式或开始菜单打开 Mineradio。

不要下载 `Source code`、`.blockmap`、`latest.yml`。这些不是普通用户直接运行的安装包。

## 2. 登录音乐账号

打开播放器后，可以在账号/登录入口选择平台。

| 平台 | 标签 | 说明 |
| --- | --- | --- |
| 网易云音乐 | `NE` | 原项目已有接入，可用于搜索、歌单和播放。 |
| QQ 音乐 | `QQ` | 原项目已有接入；如果扫码状态异常，可以稍后重试或改用其它音源。 |
| 酷狗概念版 | `KG` | 扩展版新增，支持扫码登录、搜索、播放、歌单、红心、收藏、歌词和评论。 |
| 普通酷狗音乐 | `KGM` | 扩展版新增，和酷狗概念版分开登录、分开保存状态。 |
| 汽水音乐 | `QS` | 当前主要支持分享歌单导入；不等于完整汽水账号客户端。 |

建议需要哪个平台就登录哪个平台。多个平台同时登录后，可以在“我的歌单”里切换不同平台的歌单。

## 3. 搜索和播放

顶部搜索框支持按平台筛选：

- `All`：混合搜索多个平台。
- `NE`：只搜网易云。
- `QQ`：只搜 QQ 音乐。
- `KG`：只搜酷狗概念版。
- `KGM`：只搜普通酷狗音乐。
- `QS`：用于汽水分享链接或汽水导入结果。

酷狗系歌曲会按实际播放来源显示音质。酷狗概念版和普通酷狗音乐目前使用四档文案：

- `Hi-Res音质`
- `无损音质`
- `高品音质`
- `标准音质`

如果请求高音质但平台实际只返回较低码率，播放器会以实际返回结果为准。

## 4. 导入汽水音乐歌单

1. 在汽水音乐里复制分享歌单链接。
2. 回到 Mineradio，把链接粘贴到搜索框。
3. 播放器会尝试解析分享页并生成本地“汽水音乐歌单”。
4. 导入成功后，可以在“我的歌单”里直接打开这个本地歌单。

注意：汽水音乐当前是阶段性接入。部分汽水音源可能返回浏览器不能直接播放的加密音频，这时播放器会尝试换源到其它平台。如果没有可匹配版本，仍可能无法播放。

## 5. 管理歌单和排序

打开“我的歌单”后，可以进入歌单详情页。

当前支持的常用排序：

- 收藏时间：更适合看最近加入的歌。
- 歌手名：适合按歌手整理。
- 本地播放次数：使用本播放器里的本地播放记录，不等于音乐平台官方全量播放次数。

酷狗平台可能返回部分平台播放次数，但它不是完整终身听歌数据库。没有返回的数据会显示为暂无或 0，不应理解为你从未听过。

## 6. 收藏、红心和新建歌单

酷狗概念版和普通酷狗音乐都已接入写入操作：

- 红心喜欢
- 新建歌单
- 收藏当前歌曲到歌单

这些操作会修改你的真实音乐平台账号。使用前请确认当前登录的是你自己的账号。

## 7. 个性化 Home 海报

首页左侧有“我的音乐海报”：

- `换图片`：选择一张本地图片作为海报背景。
- `用当前封面`：把正在播放的歌曲封面作为海报图。
- `改文案`：在页面内编辑一句自己的文字。
- `重置`：恢复默认海报。

图片和文案保存在本机浏览器本地存储里，不会上传到仓库或外部服务。

## 8. DIY 视觉和形态工坊

视觉控制台里有“我的作品”和“形态工坊”。

### 我的作品

“我的作品”用于保存一整套视觉配置，例如背景、颜色、粒子、歌词、镜头和部分交互参数。

支持：

- 新建作品
- 保存当前配置
- 复制作品
- 导入 `.mineradio-visual.json`
- 导出 `.mineradio-visual.json`

### 形态工坊

“形态工坊”用于创建自己的 3D 粒子形态。

基础流程：

1. 打开视觉控制台。
2. 进入“预设”页。
3. 点击“创建形态”。
4. 选择点、线、环、曲线环、面、螺旋、波、隧道、球面、星尘、卡片等基础粒子。
5. 拖动舞台上的手柄调整位置和角度。
6. 用右侧滑条调整数量、大小、前后厚度、透明度和随音乐动。
7. 默认是静态编辑；需要看动态效果时再打开“律动预览”。
8. 保存为“我的形态”后，可以像内置预设一样应用到播放器。

支持导入/导出 `.mineradio-shape.json`。公开版不会内置二创作者个人实验形态，用户形态需要自己创建、保存或导入。

## 9. 常见问题

### 下载被浏览器拦截怎么办？

确认文件来自本仓库 Release 后，可以在浏览器下载列表里选择 `保留` 或 `仍要保留`。如果系统或杀毒软件明确报高危，不要强行运行，建议重新下载并反馈截图。

### 为什么汽水歌单能导入，但有些歌实际播放走了别的平台？

因为部分汽水音源返回的格式浏览器不能直接播放。播放器会尽量用已登录且可播放的平台匹配同名歌曲，这是当前阶段的降级策略。

### 为什么显示的播放次数和手机 App 不一样？

本地播放次数只统计你在 Mineradio 里播放过的次数。部分平台播放次数来自平台接口能返回的范围，不保证覆盖账号完整历史。

### 酷狗音效在哪里？

当前没有接入酷狗官方音效。公开版不会用本地 WebAudio 模拟效果冒充官方音效。
